Calculating the date of Easter with Python

from dateutil.easter import easter  
print(easter(2010))  
2010-04-04

The function also has an argument, where you can specify the type of Easter calculation you would like:

1 = Easter Julian
2 = Easter Orthodox
3 = Easter Western 

print(easter(2010, 1))
2010-03-22  
print(easter(2010, 2))
2010-04-04  
print(easter(2010, 3))  
2010-04-04
